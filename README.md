# How to use the Task App
1. Set it up
 - clone the code from my GitHub.
     https://github.com/Guddu2055/Todo-Application.git

 - Open your terminal in the project folder and type: npm install.

 - Start the app by typing: npm run dev.

2. Log in
 - Open the app in your browser.

 - Use my account: * Email: guddu2055@gmail.com

 - Password: Aabc123#

 - Or: Create your own account with your email.

3. Navigation
 - After you login, you will see a "Go to Todo" button.

 - Click it to see your task list.

4. What you can do
 - Add: Click the "Add New Task" button to make a new one.

 - Edit: Click the "Edit" button to change a task you already made.

 - Delete: Click the "Delete" button to remove a task.

 - List: View all your tasks in one simple table.
